# Raylib SNAKE GAME

Made just for the sake of learning raylib

## How to use
You need to provide your Own Raylib.h and raylib libraries, and compilie using
`g++ main.cpp Snake.cpp Food.cpp -I include/ -L lib -lraylib -lopengl32 -lgdi32 -lwinm`
